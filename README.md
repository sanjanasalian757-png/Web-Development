# Glassmorphism Login Page

A modern, responsive, and visually striking login page featuring a **glassmorphism** design effect. This project was built using semantic HTML5 and custom CSS3 properties, featuring icons from Boxicons and typography from Google Fonts.

## 🚀 Features

* **Glassmorphism UI:** Features a sleek, frosted-glass effect using CSS `backdrop-filter` and semi-transparent borders.
* **Fully Responsive:** Optimized to look great on desktops, tablets, and mobile screens.
* **Modern Typography & Icons:** Built using the clean *Poppins* font family and crisp *Boxicons*.
* **Interactive Form Elements:** Form inputs include custom focus states, floating icons, and smooth hover transitions on the submit button.

---

## 🛠️ Built With

* **HTML5:** Structured semantic markup.
* **CSS3:** Custom styling, Flexbox layout, glassmorphism filters, and smooth transition animations.
* **Boxicons:** Scalable vector icons for the user and lock symbols.
* **Google Fonts:** Clean integration of the *Poppins* font family.
